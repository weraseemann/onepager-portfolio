# Portfolio Website – Development Plan

## Project Overview
Development of a personal portfolio website using React and Tailwind CSS to present projects, skills, and professional experience.

## Goals
- Showcase development projects
- Provide contact options for employers
- Demonstrate React and Tailwind skills
- Create a responsive and modern UI

## Technology Stack
- React
- Vite
- Tailwind CSS
- Git / GitHub
- Vercel (deployment)

## Website Structure
- Home / Hero
- About
- Skills
- Projects
- Experience
- Contact
- Footer
